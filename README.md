# Equipment Booking & Inventory Management System
> Масштабируемый веб-сервис для бронирования оборудования с ролевой моделью доступа и интеграцией поисковых движков.

## 🎯 Обзор
Проект представляет собой автоматизированную систему управления материальными ценностями. Реализована сложная логика бронирования, фильтрации и аналитики с использованием современного стека Python-разработки.

## 🛠 Технологический стек
* **Core:** Python 3.x, Flask
* **Database:** SQLAlchemy (ORM), Flask-Migrate (Alembic)
* **Search & Analytics:** Elasticsearch
* **Security & Auth:** Flask-Login, WTForms (валидация), Werkzeug (хеширование).
* **Templating:** Jinja2, HTML5, CSS3.

---

## Функциональные возможности

### User Management & Auth
* **Secure Auth:** Регистрация и вход в систему с обязательным хешированием паролей.
* **Role-Based Access Control (RBAC):** Реализованы три уровня полномочий:
  * **Level A (Viewer):** Только просмотр списка оборудования.
  * **Level B (User):** Просмотр + бронирование и отмена своих заявок.
  * **Level C (Admin):** Полный контроль + отмена любых бронирований в системе.

### Booking Logic & Database
* **Database Integration:** Разработка и заполнение БД оборудования (номенклатура, статусы, локации).
* **Booking Engine:** Логика аренды с учетом расписания, длительности «аренды» и текущего статуса оборудования.
* **User Prioritization:** Учет приоритетов пользователей при распределении ресурсов.
* 
### Search & Filtering
* **Advanced Search:** Настройка поиска и фильтрации по наличию оборудования, его вместимости и физическому местоположению (локациям).
